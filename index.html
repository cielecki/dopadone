How it works

Every engagement trick that platforms use to hijack your attention, pointed at your own goals.

The Slot Machine

Hit Start Run and a weighted random selection picks your next task from Todoist, GitHub issues, markdown vault tasks, habits, and calendars. You didn't choose it — the machine did. That reframe cuts through decision paralysis.

Progress Over Completion

The biggest button on the screen is Progress, not Done. Any movement counts — start it, delegate it, research it. Five minutes is all you need.

One Task at a Time

Your anxious mind gets one thing to focus on. Not a list of 47 things generating guilt. One task, one timer, one decision.

Dopamine, Reclaimed

Platforms engineer dopamine loops to keep you scrolling. DopaDone uses the same mechanics — XP, streaks, escalating rewards — but points them at your own tasks. The dopamine hits, but it serves you.

Tasks, Habits & Calendars

Todoist, GitHub issues, Obsidian vault, recurring habits, calendar prep — every source feeds into one weighted pool. Mix and match freely.

Open Source, Your Data

Everything is stored in markdown files — human-readable, AI-processable, portable. The source code is open, so you can fork it and build your own variant. No lock-in, ever.
